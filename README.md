# Workload Stress Detector

## Description
The Workload Stress Detector is a Python desktop application developed using Tkinter. It helps analyze employee workload and estimates stress levels based on working hours, meetings, break hours, and overtime.

## Features
- Employee workload analysis
- Stress score calculation
- Stress level recommendation
- Save employee records to CSV
- Display stress graph
- Easy-to-use graphical interface

## Technologies Used
- Python
- Tkinter
- Pandas
- Matplotlib

## Project Structure

```

```
